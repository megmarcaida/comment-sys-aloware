# comment-sys
 
## Installation
##### composer install
##### npm install 

## Migration
##### create .env file 
##### DB_CONNECTION=mysql
##### DB_HOST=127.0.0.1
##### DB_PORT=3306
##### DB_DATABASE=comment_sys
##### DB_USERNAME=root
##### DB_PASSWORD=
##### create db 'comment_sys'
##### php artisan migrate

## Running
##### npm run dev

## Unit Testing
##### create `Unit` folder on `tests` folder
##### php artisan test
